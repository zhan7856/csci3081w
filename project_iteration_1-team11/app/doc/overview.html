Introduction

Vehicles

The Visual Transit Simulator models vehicles (buses and trains) and passengers around the University of Minnesota campus. These vehicles provide service for a line which is made up of inbound and outbound routes. Vehicles go along a route, make stops, and pick up/drop off passengers.

Simulation

The simulation is made up of a visualization and simulation module. The visualization part shows what is happening while the simulation part performs the actual simulation. The configuration is stored in file called config.txt which defines the vehicle lines, stops, and probabilities for passengers to appear at each stop. The simulation can also be paused!
